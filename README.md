# Portfolio Web

Portfolio personal desarrollado con React, Vite y TypeScript, con foco en rendimiento, arquitectura escalable, experiencia visual moderna e internacionalización.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- react-i18next

## Objetivos del proyecto

- Mostrar proyectos, experiencia y recorrido profesional
- Mantener una arquitectura limpia y escalable
- Priorizar rendimiento y buena experiencia en mobile
- Permitir contenido multilenguaje
- Facilitar despliegue en Vercel

## Estructura

```text
src/
  app/        # providers, router, estilos globales
  content/    # datasets y contenido por idioma
  entities/   # tipos y piezas reutilizables del dominio
  features/   # páginas y secciones del portfolio
  shared/     # hooks, config, utilidades y UI reutilizable
```
